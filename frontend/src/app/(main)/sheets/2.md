The page displays sheets as **fullвҖ‘width stacked cards** (vertical list). Each card shows: sheet name, description, owner, participant avatars (up to 4, then `+X`), participant count, and an action button (вҖңJoinвҖқ or вҖңViewвҖқ). A top bar provides search, filter, sort, and a вҖңCreate New SheetвҖқ button.

Responsive behaviour is described below the wireframe.
